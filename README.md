# Сервіс оренди павербанків


Система для оренди портативних зарядних пристроїв.

## Основний функціонал
- Пошук найближчих станцій
- Оренда павербанка через QR-код
- Автоматичний розрахунок вартості
- Повернення на будь-яку станцію

## Технології
- Node.js
- Docker
- PostgreSQL
- GitHub Actions

## Швидкий старт

```bash
git clone https://github.com/ТВІЙ_НІК/powerbank-rental.git
cd powerbank-rental
npm install
npm run dev
